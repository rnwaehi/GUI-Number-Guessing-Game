# GUI-Number-Guessing-Game
A number guessing game made using Tkinter GUI

## Description
This is a number guessing game made with 
Python's built-in library for creating 
graphical user interfaces (GUIs) which is called Tkinter. The games gives the user
feedback that lets them know if the number they guesses is either too high or too low 
or if it is correct. It also provides a number of attempts for the user.

## Gameplay Walkthrough
1. Main Menu: Before the user able to play the game, they will be prompted
with a menu display that allows the user to choose a difficulty. The options include:
Easy, Medium and Hard.
2. Make a Guess: After choosing the difficulty, the player inputs
their guess in the range specified.
3. Attempts And Feedback: If the user enters their guess, they will be a
message that tells them if their is too high or too low, and as long as
they dont guess the correct number their attempt counter will decrease.
4. Game duration: The user can keep guessing till they run out of attempts
or till they guess the right number.
5. Win or Lose Conditions: If the user guesses correctly, the game congratulates
the player by showing victory message. However, if they ran out of attempts
then a Game over message is shown with the random number that was hidden from
the user.
